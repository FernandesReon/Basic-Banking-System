ABOUT US

Hello! and Greetings everyone. I am Reon Fernandes, from Mumbai, India. I have made this simple website that is based on concept of illustrating a simple banking system. This is my task 1 of Web Development internship at Spark Foundation.

To Know more about Spark Foundation, Click on the link. Check out their GRIP Program and LinkedIn Page.

So basically the objective of this project is to make a basic banking system, wherein independent user can perform money transaction. Also, the user can check the transaction history at any given point of time.

While making this project I learned a lot about Web Development, getting some hands-on experience, also it was fun making this project, which challenged me to improve, explore new stuff and polish my skills.

In this project I have used various Frontend Technologies such as [HTML, CSS] and Backend Technologies such as [Java] and concepts related to Java like JDBC(for database connectivity, storing data and accessing it), JSP (for displaying dynamic content) and Servlet (for server side logic of project). and for storing data I have used SQL Database -- MySQL.
